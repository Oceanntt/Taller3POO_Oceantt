Descripción del proyecto: Aplicación de consola en Java para la gestión y análisis de magos y hechizos. Permite administrar magos y hechizos mediante archivos de texto, agregando, modificando y eliminando registros. Además, incluye funcionalidades de análisis para evaluar el puntaje de los magos según los hechizos que poseen.

Estructura:

Main.java = Clase principal y menú de la aplicación.
Mago.java = Clase del dominio que representa un mago.
Hechizo.java = Clase base para todos los hechizos.
Tierra.java, Fuego.java, Agua.java, Planta.java = Clases derivadas de Hechizo que representan los distintos tipos de hechizos.        

Archivos de texto:

Magos.txt
Hechizos.txt

Instrucciones de ejecución:

Abrir el proyecto en un compilador o IDE compatible con Java.
Verificar que los archivos Magos.txt y Hechizos.txt se encuentren dentro de la carpeta del proyecto.
Ejecutar la clase principal Main.java.
Seleccionar el modo de uso:
Analista
Administrador
Seguir las instrucciones mostradas en consola para gestionar o analizar la información de magos y hechizos.
